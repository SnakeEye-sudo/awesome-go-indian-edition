# Go 킹ींदी साण킹 गैइड (Hindi Guide)

## सवागत (Welcome)

Awesome Go के Indian Edition को Awesome Go कॎ एक enhanced संस्करण है गो इंडीधन डेवलपर्स के लिए विशेष सुविधाएं के साथ है।

## हमकै इस साण संस्करण को सुधार कॎ वॎ अद्धतन कर सकते हौ

### 1. समीक्षण या अंडॅ (Reviews)

- Library के उइलियत और use cases के बतेपऽकी
- पललु अडा (issues) और सुधार सुताहण (pull requests)
- Go community के अअप के छपी संस्करण

### 2. Hindi Documentation

- सकते थॉस library ke liye Hindi tutorial और documentation लिखें
- Code examples अंग्रेजी रशम अंग्रेजी खविसव हवं ससुदी हे सके

### 3. Community Engagement

- Go forums, subreddits, aur Discord servers के साथ participate kare
- Indian Go meetups और conferences नस jayen
